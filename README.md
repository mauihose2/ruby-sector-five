## Sector-Five Setup

### Install RVM to manage rubies (not in scope)

* https://rvm.io/

### Clone the repo
* git clone https://github.com/mauihose2/ruby-sector-five.git
* cd into the repo

### Install bundler
* gem install bundler

### Install gosu
* gem install gosu

### Start the game
* cd bin
* ./play_game
